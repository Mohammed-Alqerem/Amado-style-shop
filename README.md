**Live Demo:**

---

## Project Description

A clean, simple, and responsive e‑commerce template that showcases modern-product layouts (Modern Chair) and includes essential pages such as **Home, Shop, Product, Cart, and Checkout**. The project is built using **HTML**, **CSS**, and **Bootstrap**.

---

## Features

* Fully responsive design using Bootstrap
* Navigation bar with links to main pages
* Product cards with images, titles, and pricing
* Newsletter subscription placeholder
* Cart and checkout UI structure

---

## Project Structure (Example)

```
amado-style-shop/
├─ index.html            # home
├─ shop.html             # product listing
├─ product.html          # product details
├─ cart.html             # shopping cart page
├─ checkout.html         # checkout flow
├─ assets/
│  ├─ css/
│  ├─ js/
│  └─ images/
└─ README.md
```

---

## Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/mohammed-alqerem/Amado-style-shop.git
   ```
2. Open `index.html` directly in your browser **or** use the VS Code *Live Server* extension for a better preview.

---

## Customization

* Modify text and images inside the HTML files and the `assets/images` folder.
* Edit colors and typography in the CSS files.
* Add JavaScript for interactive features like cart handling using **localStorage** or integrating with a backend API.
